Вводите любой запрос вида:
{
    "weatherDate":"11-22-33"
}

И в ответ вернется ваш прогноз, так как, соответственно, такого прогноза в базе нет.
Для проверки, ищутся ли данные в базе, я предварительно загрузил одно значение в базу,
чтобы его вывести, введите:

{
    "weatherDate":"66-66-66"
}
